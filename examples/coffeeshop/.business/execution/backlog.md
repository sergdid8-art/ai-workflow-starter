# Бэклог Coffee Place

## Сейчас актуально (1-2 недели)

- Найти поставщика растительного молока с ценой ниже -10% к текущей
- Обновить меню на стенде в точке №3 (появились новые позиции)
- Собрать отзывы клиентов для лендинга подписки

## Когда-нибудь (месяцы)

- Провести мастер-класс по латте-арту для клиентов (в точке №1)
- Запустить свой подкаст про кофе-культуру Казани
- Сделать коллаб с местным шоколатье на эксклюзивные десерты
- Летний фестиваль колд-брю на набережной

## Архив (не делаем)

- Франшиза — решили не масштабироваться через франчайзи (теряем контроль качества)
- Доставка кофе на дом по подписке — расчёты показали что невыгодно при наших объёмах
