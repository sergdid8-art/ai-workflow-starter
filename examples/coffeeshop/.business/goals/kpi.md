# Ключевые метрики

## Еженедельные

- **Выручка по точке** (3 точки отдельно)
- **Средний чек** — цель 400 ₽ к концу года
- **Количество транзакций** через приложение vs касса
- **Новые регистрации в приложении**

## Ежемесячные

- **Повторные клиенты** (процент тех кто пришёл 2+ раза) — цель 65%
- **NPS по отзывам в 2ГИС + Яндекс.Картах** — цель 4.7+/5
- **Фудкост** (себестоимость продуктов к выручке) — цель 28-32%
- **Labor cost** (зарплаты к выручке) — цель 30-35%
- **DAU приложения**
- **Отток бариста** (текучка за месяц)

## Ежеквартальные

- **Прибыльность каждой точки** (чистая прибыль за квартал)
- **LTV клиента** (средняя выручка с клиента за 90 дней)
- **Выручка новой подписки** (начиная с Q2)

## Где данные

- Касса и эквайринг — Эвотор + Альфа-Банк (ежедневно выгружается в Google Sheets)
- Приложение — Firebase Analytics (автоматически)
- Зарплаты — 1С Бухгалтерия
- Отзывы — Яндекс.Карты API + 2ГИС API
- Общий дашборд — в Metabase, смотрю 2 раза/неделю
