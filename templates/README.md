# templates/ - образцы файлов для генерации

Агент берёт файлы отсюда как образцы, когда создаёт конкретные файлы под проект пользователя.

## CLAUDE.md.tmpl

Шаблон для генерации корневого `CLAUDE.md`. Содержит плейсхолдеры:
- `{{PROJECT_NAME}}` — имя проекта
- `{{PROJECT_ONE_LINER}}` — короткое описание проекта
- `{{STACK}}` — технический стек
- `{{LANGUAGE_RULE}}` — язык общения

Агент заполняет плейсхолдеры и записывает итог в корень как `CLAUDE.md`.

## agent-settings.example

Пример локальной конфигурации ограничений и рабочих настроек для конкретного проекта.
