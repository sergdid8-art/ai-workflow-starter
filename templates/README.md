# templates/ - образцы файлов для генерации

Клод берёт файлы отсюда как образцы, когда создаёт конкретные файлы под проект пользователя.

## CLAUDE.md.tmpl

Шаблон для генерации корневого `CLAUDE.md` на шаге 9a AUTOPILOT. Содержит плейсхолдеры:
- `{{PROJECT_NAME}}` - имя проекта
- `{{PROJECT_ONE_LINER}}` - одно предложение о чём проект
- `{{STACK}}` - технический стек
- `{{LANGUAGE_RULE}}` - на каком языке Клоду общаться с пользователем

Клод заполняет плейсхолдеры ответами из интервью и записывает в корень как `CLAUDE.md`.

## claude-settings.json.example

Образец `.claude/settings.json` после прохождения шагов 4 и 5 AUTOPILOT (hooks + 4 запрета безопасности).

**Это не валидный рабочий файл** (Claude Code может ожидать свою версию формата). Образец даёт Клоду понимание структуры, в реальном файле Клод использует актуальный формат Claude Code.

Структура:
- `hooks.Stop` - звук при завершении задачи
- `hooks.Notification` - звук при запросе разрешения
- `permissions.deny` - запреты опасных команд

Кросс-платформенная команда звука использует `afplay` (Mac) с fallback на PowerShell `console.beep` (Windows) и терминальный bell `\a`.
