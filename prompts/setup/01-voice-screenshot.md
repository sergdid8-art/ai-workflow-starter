# Голосовой ввод + скриншоты в буфер

**Когда брать:** в самом начале работы, до того как начнёшь много писать промпты.
**Что делает:** ускоряет тебя в 3-5 раз. Голосом диктуешь быстрее, скриншоты показывают Клоду контекст вместо 1000 слов.
**Результат:** работает голосовой ввод по горячей клавише + скриншот сразу в буфер обмена.

## Часть 1. Голосовой ввод

Wispr Flow (лучшее на рынке, платный) - [wisprflow.ai](https://wisprflow.ai). Есть пробный период.
Super Whisper (Mac) - [superwhisper.com](https://superwhisper.com).
Бесплатный fallback: Mac - System Settings → Keyboard → Dictation. Windows - `Win + H`.

### Промпт

```prompt
Помоги мне установить Wispr Flow (или Super Whisper, если я скажу) на [Mac/Windows].

1. Дай прямую ссылку на скачивание.
2. Объясни простыми словами, что сделать после скачивания: куда перетащить, какие разрешения дать (микрофон, accessibility).
3. Помоги настроить горячую клавишу: одно нажатие - начал диктовать, отпустил - текст вставился в активное окно.
4. Продиктуй короткий чек-лист как проверить что всё работает.

Действуй по одному шагу. Жди моих ответов.
```

**Проверка:** открой любой текстовый редактор, нажми горячую клавишу, скажи «привет, проверка связи» - должен появиться текст.

## Часть 2. Скриншот в буфер обмена

Скриншот должен **сразу** попадать в буфер (не на рабочий стол). Дальше `Ctrl/Cmd + V` в чат с Клодом.

### 🍎 Mac

`Cmd + Shift + Ctrl + 4` - скриншот области в буфер.

Если неудобно - переназначь:
1. System Settings → Keyboard → Keyboard Shortcuts → Screenshots
2. «Copy picture of selected area to the clipboard»
3. Кликни и нажми свою комбинацию

### 🪟 Windows

`Win + Shift + S` - Snipping Tool. Выделил - в буфере.

**Проверка:** сделай скриншот → открой Claude Code → `Ctrl/Cmd + V` → должен появиться в чате.
