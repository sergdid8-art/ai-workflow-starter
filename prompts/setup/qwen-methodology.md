# Qwen methodology setup

## Цель

Подготовить рабочую методологию для Qwen, чтобы будущие проекты реализовывались быстрее, дешевле и безопаснее.

## Что сделать

1. Прочитай файл QWEN.md в корне проекта.
2. Сохрани его как основную инструкцию для работы с Qwen в этом репозитории.
3. При необходимости адаптируй под конкретный проект: стек, язык, правила команд и форматы работы.
4. Для каждой новой задачи сначала проверь:
   - есть ли контекст в `.business/`;
   - есть ли план в `plans/`;
   - есть ли похожий промпт в `prompts/`.
5. Если задача сложная, сначала предложи краткий план, а уже потом приступай к реализации.

## Критерии успешной работы

- меньше бессмысленных действий;
- меньше затрат на лишние итерации;
- меньше рисков из-за неполного контекста;
- более понятный и повторяемый процесс.
