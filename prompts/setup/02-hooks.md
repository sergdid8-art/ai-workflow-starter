# Звуковые хуки (hooks)

**Когда брать:** сразу после онбординга, до начала реальной работы.
**Что делает:** включает звуковое уведомление когда Клод закончил задачу или ждёт твоего разрешения. Ты не пропустишь момент когда можно продолжать.
**Результат:** `.claude/settings.json` с двумя hooks (Stop + Notification).

## Проблема которую решает

Ты запустил задачу, ушёл за чаем. Клод закончил через 2 минуты и ждёт твоего ответа. Ты думаешь что он ещё работает. Полчаса потеряно.

## Промпт

```prompt
Настрой мне в .claude/settings.json звуковые уведомления (hooks):

1. Звук при завершении задачи (Stop hook) - чтобы я слышал, когда ты закончил работу.
2. Звук при запросе разрешения (Notification hook) - чтобы я слышал, когда ты ждёшь моего «да/нет».

Используй системный звук моей операционной системы.

За образцом структуры посмотри templates/claude-settings.json.example в этом проекте - там пример hooks для Mac/Windows с fallback.

Перед изменением файла - покажи мне текущее содержимое .claude/settings.json. После изменения - покажи итоговый файл и объясни, что добавил.
```

## Проверка

Задай любую микрозадачу («посчитай 2+2 и всё»). Когда Клод закончит - должен пикнуть.

Если не пикает:
- Проверь что звук не отключён в системе
- Проверь что `.claude/settings.json` содержит секцию `hooks`
- Перезапусти VS Code
