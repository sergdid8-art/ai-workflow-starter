# Code of Conduct

## Что мы хотим

Этот проект открыт для всех — независимо от опыта, национальности, пола, возраста, религии и других личных характеристик. Мы хотим чтобы каждый участник чувствовал себя уважаемым и безопасно вносил свой вклад.

## Что приветствуется

- Уважительная и конструктивная критика
- Помощь новичкам
- Обмен опытом и кейсами использования шаблона
- Сообщения об уязвимостях и багах
- PR на улучшения промптов, документации, переводов
- Обратная связь, даже негативная — если аргументированная

## Что недопустимо

- Оскорбления, дискриминация, травля
- Публикация чужих приватных данных (адресов, телефонов, email, ПДн клиентов) без согласия
- Флуд, спам, реклама несвязанных продуктов
- Троллинг, срачи на политические темы
- Попытки использовать проект для распространения вредоносного кода

## Применение

Нарушения разбираются мейнтейнером. Возможные меры:
1. **Предупреждение** — первое нарушение, разъяснение правил
2. **Временный бан** (7-30 дней) — повторное или грубое нарушение
3. **Постоянный бан** — угрозы, травля, вредоносный код

## Куда жаловаться

- Через Issues (если нарушение публичное и не содержит ПДн)
- Через [GitHub Security Advisory](https://github.com/artemiimillier/claude-code-starter/security/advisories/new) (приватно)

Мы рассмотрим любое обращение в течение 7 дней.

## Атрибуция

Основано на [Contributor Covenant](https://www.contributor-covenant.org/) v2.1.
